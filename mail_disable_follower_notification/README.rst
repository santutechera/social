This module disables, by default, the option to send a notification
email when adding followers to a record in the chatter.

**Table of contents**

.. contents::
   :local:

Usage
=====

To use this module, you need to:

1. Go to any record form view
2. On the chatter, click on the "Followers" icon, the one with person's
   shape
3. Click "Add Followers"
4. You will see that the "Send Notification" option is disabled
